## 1.2.字段查询
```
GET test_search_index/_search?q=username:alfred
```
只查询`username`字段包含`alfred`

### 1.2.1.双引号的语义
```
GET test_search_index/_search?q=username:"alfred way"
```
将 ``alfred way``当作一个整体做查询。上面的实例中只返回一条结果：

### 1.2.2.小括号的语义
```
GET test_search_index/_search?q=username:(alfred way)
```
是``或``的意思，类似``shell``中的``|``
