## 02.矩阵与矩阵运算

```python
A = np.arange(4).reshape(2, 2)
B = np.full((2, 2), 10)
A # array([[0, 1],
  #        [2, 3]])
B # array([[10, 10],
  #        [10, 10]])

A + B
array([[10, 11],
       [12, 13]])

A * B
array([[ 0, 10],
       [20, 30]])

A.dot(B)
array([[10, 10],
       [50, 50]])

A.T  # 转置
array([[0, 2],
       [1, 3]])
```



## 03.矩阵和向量运算

```python
v = np.array([1, 2])
A = np.arange(4).reshape(2, 2)

V # array([1, 2])
A # array([[0, 1],
  #        [2, 3]])
    
v + A
array([[1, 3],
       [3, 5]])

v * A
array([[0, 2],
       [2, 6]])

v.dot(A)
array([4, 7])

A.dot(v)
array([2, 8])
```



## 03.矩阵的逆

```python
np.linalg.inv(A)
array([[-1.5,  0.5],
       [ 1. ,  0. ]])
```
