## Java 强引用、软引用、弱引用、虚引用的区别
- 强应用：同原始应用概念，永远不会被回收的对象
- 软引用：有用但非必需，当内存不够时回收，如果还不够抛出 OutMemoryException
- 弱引用：活不过下次垃圾回收
- 虚引用：无法通过虚引用获取该对象，唯一的作用和目的就是当对象回收时收到系统通知
